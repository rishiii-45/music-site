# 🎵 DopeTracks – Music Playlist Website

DopeTracks is a responsive, dark-themed music playlist website built using HTML, CSS, and JavaScript. It allows users to explore songs, view playlists, and enjoy smooth UI interactions.


## Live Demo
https://rishiii-45.github.io/music-site/


## Tech Stack

- HTML5  
- CSS3  
- JavaScript  


## Features

- Artist-based playlists  
- “All Tracks” section  
- Dark-themed modern UI  
- Smooth navigation and hover effects  
- JavaScript-based audio playback
